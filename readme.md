### List Endpoints:

* Store:
- Store Init
- Store Information
- Create Store

* Sales Report:
- Discount Spent
- Total Sales
- Total Transaction (count transaction)
- Voided Transaction
- List Transaction (for table)

* Outlet Report
- Total Sales
- Total Transaction
- Product Sold
- Progress Outlet with date range (multiple outlet)
- List Outlet with Progress data result

* Payment Report
- Sales (Inc Tax)
- Sales (Exc Tax)
- Discounts
- Void
- List payments with Progress data result
